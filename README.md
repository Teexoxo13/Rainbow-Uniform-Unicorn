# Contractor Management System - PLO Assessment

Project Overview:
This project implements a structured and object-oriented computer solution for managing contractor and subcontractor data. It demonstrates the ability to design class hierarchies, implement inheritance, and apply shift-based pay algorithms.

Key Features:
*   **Inheritance:** Uses a base `Contractor` class and a derived `Uniform` class.
*   **Encapsulation:** Private member variables with public accessors and mutators.
*   **Algorithm Implementation:** Automatically calculates a 3% pay differential for night shift employees.
*   **Dynamic Instance Creation:** Allows users to input and store multiple subcontractor records.

Tech Stack:
*   **Language:** Java
*   **Version Control:** Git/GitHub
